#  文件夹命名不需添加项目前缀，但文件夹内的文件则需要

【+文件夹  -文件】

文件夹及文件命名规则如下：

+ JAFrameWork
    +Main
        +WeddingTime  //项目主功能
            -WeddingTimeMainController
            +WeddingTimeReplyTime //子功能
                -WTReplyTimeController




### 注释规则
一、采用系统提供的方法注释： 快捷键 > opt+cmd+/
```
1.不带参数和返回值
/**
<#Description#>
*/

2.带返回值的
/**
<#Description#>

@return <#return value description#>
*/

3.带参数和返回值的
/**
<#Description#>

@param color <#color description#>
@return <#return value description#>
*/

```
